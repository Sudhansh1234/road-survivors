# 🏎️ Road Survivors

Top-down multiplayer survival racing game. Dodge oncoming traffic, grab power-ups,
and be the last one driving. Built with Node.js + WebSockets.

## Run locally
```bash
npm install
npm start
```
Open http://localhost:3000

## Play with friends
- **Same WiFi:** friends open `http://YOUR-LAN-IP:3000` and enter your room code.
- **Over the internet:** deploy to Render (below) for a permanent public URL.

## Deploy to Render (free, permanent URL)
1. Push this repo to GitHub.
2. Go to https://render.com → **New → Blueprint**.
3. Select this repo. Render reads `render.yaml` and deploys automatically.
4. Share the resulting `https://<name>.onrender.com` link with friends.

## Modes
- ☠️ Last Survivor · 🤝 Co-op Survive · 🏁 Distance Race

## Controls
Move: Arrows / WASD · Boost: Shift · Brake: Space · Drop oil: E · (mobile: drag + buttons)
